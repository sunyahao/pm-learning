## ADDED Requirements

### 要求：重点项目管理章节应提供记忆与复盘目录入口
第08章至第17章的章节目录下应提供 `记忆与复盘/README.md`，用于把 `核心记忆.md`、`复盘题单.md` 和 `参考作答.md` 汇总为统一入口，帮助学习者在临考速记、自测提问和核对表达之间快速切换。为兼容章节固定结构和已有相对链接，这三份规范正文可以继续保留在章节根目录，由 `记忆与复盘/README.md` 负责集中导航。

#### 场景：学习者只想集中使用速记与复盘材料
- **前提** 学习者进入第08章至第17章任一章节目录
- **当** 他只想查看临考速记、自测问题和参考答案
- **则** 能找到 `记忆与复盘/README.md`
- **并且** 能从该目录入口快速进入 `核心记忆.md`、`复盘题单.md` 和 `参考作答.md`

### 要求：重点项目管理章节学习入口应提供记忆与复盘目录
第08章至第17章的 `README.md` 学习入口中应提供 `记忆与复盘/README.md` 的入口，用于明确“速记、自测、核对答案”是一组可集中访问的学习资产。

#### 场景：读者从章节入口寻找速记和复盘材料
- **前提** 读者打开第08章至第17章任一章节目录下的 `README.md`
- **当** 他查看学习入口
- **则** 能看到 `记忆与复盘/README.md` 的入口
