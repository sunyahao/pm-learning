# 设计：重点知识领域的记忆与复盘目录入口

## 设计目标

在不破坏章节固定结构的前提下，为第08章至第17章提供统一的“速记与复盘资产入口”。

## 设计原则

1. 根目录结构保持稳定：`核心记忆.md`、`复盘题单.md`、`参考作答.md` 继续保留在章节根目录，避免已有链接失效。
2. 入口分组更清晰：新增 `记忆与复盘/README.md`，把这三份材料作为一组集中导航。
3. 速记结构统一：第09章至第17章的 `核心记忆.md` 采用与第08章一致的增强型结构，包含快速跳转、主线、过程、易混点、案例抓手和展开阅读链接。
4. 内容仍以考试复习为导向：不把 `核心记忆.md` 写成长文，而是保留“最小够用、快速回忆”的风格。

## 目录形态

每个重点知识领域章节可新增：

- `记忆与复盘/README.md`

该文件负责汇总：

- `../核心记忆.md`
- `../复盘题单.md`
- `../参考作答.md`

## README 学习入口

章节 `README.md` 中除原有直接入口外，增加：

- `记忆与复盘：[记忆与复盘/README.md](记忆与复盘/README.md)`

这样学习者既可以直接点具体文件，也可以先进入成组入口。

## 第09章至第17章核心记忆模板

增强型 `核心记忆.md` 应优先包含：

1. 这份文档怎么用
2. 快速跳转
3. 整章主线
4. 管理基础或最该背知识块
5. 过程或主题分组
6. 最易混点
7. 案例题抓手
8. 临考前最后看什么

每个重点块下应附“展开阅读”链接，优先指向：

- `完整整理.md`
- `白话拆解.md`
- `过程场景讲解.md`
- `ITTO输出速看.md`
- `输出物/` 中对应样例文件
