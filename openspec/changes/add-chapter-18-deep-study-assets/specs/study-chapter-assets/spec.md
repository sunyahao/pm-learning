### 要求：第18章项目绩效域必须提供强化学习资产
第18章 `项目绩效域` 的章节目录下必须提供 `完整整理.md`、`精讲.md`、`复盘题单.md` 和 `参考作答.md`，用于围绕八大绩效域、预期目标、度量指标、开发方法与生命周期、规划与度量要点形成完整学习链。

#### 场景：学习者系统复习第18章
- **前提** 学习者进入 `chapters/18-项目绩效域/`
- **当** 他准备系统学习或做案例题、论文题复盘
- **则** 能找到 `完整整理.md`
- **并且** 能找到 `精讲.md`
- **并且** 能找到 `复盘题单.md`
- **并且** 能找到 `参考作答.md`

### 要求：第18章项目绩效域学习入口必须先讲解后提问
第18章 `README.md` 的学习入口中，`完整整理.md` 和 `精讲.md` 必须位于 `复盘题单.md` 之前，`参考作答.md` 必须位于 `复盘题单.md` 之后，以体现“先建立绩效域视角，再主动回忆，再核对表达”的学习顺序。

#### 场景：读者按章节入口开始学习第18章
- **前提** 读者打开 `chapters/18-项目绩效域/README.md`
- **当** 他查看学习入口
- **则** 能先看到 `完整整理.md` 和 `精讲.md`
- **并且** 之后看到 `复盘题单.md`
- **并且** 在 `复盘题单.md` 之后看到 `参考作答.md`
