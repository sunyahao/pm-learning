### 要求：第19章配置与变更管理必须提供强化学习资产
第19章 `配置与变更管理` 的章节目录下必须提供 `完整整理.md`、`精讲.md`、`复盘题单.md` 和 `参考作答.md`，用于围绕配置项、基线、配置控制、变更流程、发布回退和文档受控形成完整学习链。

#### 场景：学习者系统复习第19章
- **前提** 学习者进入 `chapters/19-配置与变更管理/`
- **当** 他准备系统学习或做案例题复盘
- **则** 能找到 `完整整理.md`
- **并且** 能找到 `精讲.md`
- **并且** 能找到 `复盘题单.md`
- **并且** 能找到 `参考作答.md`

### 要求：第19章配置与变更管理学习入口必须先讲解后提问
第19章 `README.md` 的学习入口中，`完整整理.md` 和 `精讲.md` 必须位于 `复盘题单.md` 之前，`参考作答.md` 必须位于 `复盘题单.md` 之后，以体现“先理解，再复盘，再核对”的学习顺序。

#### 场景：读者按章节入口开始学习第19章
- **前提** 读者打开 `chapters/19-配置与变更管理/README.md`
- **当** 他查看学习入口
- **则** 能先看到 `完整整理.md` 和 `精讲.md`
- **并且** 之后看到 `复盘题单.md`
- **并且** 在 `复盘题单.md` 之后看到 `参考作答.md`
