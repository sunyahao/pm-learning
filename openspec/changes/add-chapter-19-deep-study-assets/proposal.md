# 提案：补齐第19章配置与变更管理的强化学习资产

## 为什么要改

当前第19章 `配置与变更管理` 已经具备：

- `README.md`
- `主脉络.md`
- `核心记忆.md`
- `ITTO输出速看.md`
- `输出物/`
- `案例/`

但它还没有像第08章至第17章那样形成完整学习链，缺少：

- `完整整理.md`
- `精讲.md`
- `复盘题单.md`
- `参考作答.md`

这会带来两个问题：

1. 第19章虽然是案例高频联动章，但当前缺少系统讲解、主动回忆和核对答案的完整路径
2. 仓库总导航已经把第19章放在第一优先层，但其学习深度仍弱于第08章至第17章，不利于“先学重点、先拿分”的复习目标

## 要改什么

1. 在 `study-chapter-assets` 中新增第19章强化学习资产要求
2. 明确第19章 `README.md` 的学习入口顺序
3. 为第19章补齐 `完整整理.md`、`精讲.md`、`复盘题单.md`、`参考作答.md`
4. 同步更新第19章导航和资料盘点文档，确保规范与实际状态一致

## 预期效果

- 第19章形成与第08章至第17章接近的学习闭环
- 学习者可以按“完整理解 -> 教师精讲 -> 主动回忆 -> 对照修正”的顺序复习第19章
- 第19章能更好支撑整合、采购、成本、案例管理等联动题的备考
