## MODIFIED Requirements

### 要求：核心记忆文件必须服务快速复习
每个章节目录下的 `核心记忆.md` 都必须提炼该章最值得记忆的高频内容，并说明应优先跳转阅读的文件。第08章至第19章的 `核心记忆.md` 可以在不重复独立速记文件的前提下，内置“快速跳转 + 分块记忆 + 展开阅读链接”的结构，用于把主线、过程、关键输出物、高频易混点和答题抓手收束到同一份临考速记稿中。

#### 场景：上班族利用碎片时间复习某章
- **前提** 学习者只有很短的复习时间
- **当** 他打开某章节目录下的 `核心记忆.md`
- **则** 能快速看到该章最该背的概念、流程或对比点
- **并且** 能看到继续深挖时应跳转的文件路径
- **并且** 如该章采用增强型核心记忆结构，能在本页快速定位到对应重点块
