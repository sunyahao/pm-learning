# 提案：为重点知识领域增加记忆与复盘目录入口

## 为什么要改

第08章至第17章目前已经具备：

- `核心记忆.md`
- `复盘题单.md`
- `参考作答.md`

这三类材料已经形成了“临考速记 -> 主动回忆 -> 核对表达”的学习闭环，但它们目前都散落在章节根目录里，缺少一个明确的成组入口。

随着第08章 `核心记忆.md` 升级为增强型速记稿，后续第09章至第17章也会按同一规范升级。如果仍然只靠根目录平铺文件，学习者会更难一眼看出“哪些是系统讲解材料，哪些是速记与复盘材料”。

因此，需要为重点知识领域引入一个统一的 `记忆与复盘/` 目录入口，把这组三件套明确地组织起来。

## 要改什么

1. 在 `study-chapter-assets` 中增加 `记忆与复盘/README.md` 的目录入口规则。
2. 明确第08章至第17章的 `记忆与复盘/README.md` 用于汇总 `核心记忆.md`、`复盘题单.md` 和 `参考作答.md`。
3. 明确章节 `README.md` 学习入口中应出现该目录入口。
4. 同步把第09章至第17章的 `核心记忆.md` 升级到与第08章一致的增强型速记结构。

## 预期效果

- 学习者能把“系统讲解”和“速记复盘”两类材料分开看，入口更清晰。
- 第08章至第17章的速记稿结构更统一，便于横向复习十大知识领域。
- 不需要打断现有根目录固定结构和历史相对链接。
