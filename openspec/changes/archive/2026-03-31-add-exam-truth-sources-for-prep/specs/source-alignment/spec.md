### 要求：历年真题必须作为正式题型来源
仓库必须将 `2015-2024论文写作真题/`、`2015-2024年案例分析历年真题/` 和 `2015-2024年选择题真题/` 视为正式题型来源，用于支撑论文趋势判断、案例题型归类、上午题高频点提炼和训练材料校准。

#### 场景：维护者需要判断某类题应该依据哪些来源
- **前提** 维护者正在编写论文趋势、案例索引、选择题高频点或冲刺训练材料
- **当** 他查看来源规则
- **则** 能确认历年真题属于正式事实来源
- **并且** 能区分论文、案例和选择题各自对应的真题目录

### 要求：不同派生资产必须匹配对应题型来源
章节学习资产继续以 `高项-三色笔记.pdf` 为主来源；论文趋势、案例索引、上午题训练和押题判断等跨章节材料，必须优先使用与之匹配的历年真题来源，不得只凭讲义或范文替代真题依据。

#### 场景：编写一份论文趋势或案例训练材料
- **前提** 维护者正在编写 `guides/`、章节 `案例/` 或其他训练型文档
- **当** 他需要引用“近几年考了什么”“常见问法是什么”“标准答案怎么看”
- **则** 会优先查对应题型的历年真题
- **并且** 不会仅凭辅助讲解资料直接下结论

### 要求：真题引用必须标明年份批次和答案依据
凡是基于历年真题生成的趋势、索引、训练或讲解材料，都必须至少标明题目的年份、上下半年与批次；若同时存在 `无答案版` 与 `有答案版`，应明确题干依据和答案依据分别来自哪个版本。

#### 场景：维护者整理某年案例题或选择题答案
- **前提** 某道题同时存在 `无答案版` 和 `有答案版`
- **当** 维护者在派生文档中引用题目或答案
- **则** 能明确说明题干取自哪个版本
- **并且** 能明确说明答案校对依据是官方有答案版还是仓库自行整理
