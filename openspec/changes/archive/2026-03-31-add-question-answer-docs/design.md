# 设计：参考作答与入口顺序

## 目标

让重点章节形成一套完整的“讲解 -> 提问 -> 核对答案”学习链条。

## 设计原则

### 1. 答案与问题一一对应

每份 `参考作答.md` 按现有 `复盘题单.md` 的编号顺序组织，便于学习者逐题核对。

### 2. 答案强调作答要点

答案不追求冗长转述，而是给出适合复述、适合案例作答的参考表达。

### 3. 入口顺序服务学习节奏

重点章节的学习入口按以下顺序组织：

1. `完整整理.md`
2. `精讲.md`
3. `复盘题单.md`
4. `参考作答.md`
5. `主脉络.md`
6. `核心记忆.md`
7. `ITTO输出速看.md`

这样可以先完成输入，再进入复习思考和核对。

## 文件结构

每份 `参考作答.md` 采用统一结构：

1. 来源说明
2. 使用方式
3. 按问题编号给出参考答案
4. 必要时补充一句作答提醒
