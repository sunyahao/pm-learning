# 提案：为重点复盘题单补充答案并调整学习入口顺序

## 为什么要改

当前第08章至第17章已经提供 `复盘题单.md`，可以用于主动回忆和口头自测，但还缺少与提问一一对应的参考答案。这样会带来两个问题：

1. 学习者提问后无法快速核对自己的回答是否到位
2. 结对复习时，带问者有问题清单，但缺少统一答案框架

同时，当前部分章节 `README.md` 的学习入口将 `复盘题单.md` 放在较前位置，不符合“先看完整讲解，再通过复盘题单复习思考”的学习顺序。

因此，需要补充答案型文档，并同步调整重点章节的入口顺序。

## 本次变更内容

1. 在 `study-chapter-assets` 规范中新增“第08章至第17章必须提供 `参考作答.md`”的要求
2. 明确重点章节 `README.md` 学习入口中，`复盘题单.md` 应位于 `精讲.md` 之后
3. 为第08章至第17章补充 `参考作答.md`
4. 调整第08章至第17章 `README.md` 的学习入口顺序
