# 提案：为第11章成本管理补充工具与技术白话说明

## 为什么要改

第11章 `成本管理` 现在已经有 `完整整理.md`、`精讲.md`、`过程场景讲解.md` 和 `白话拆解.md`，但在“估算成本”这一步，学习者仍然容易卡在几种常见方法上：

1. `类比估算`
2. `参数估算`
3. `自下而上估算`
4. `三点估算`

这些术语在教材里都有定义，但仅靠一行定义，学习者仍然不容易真正理解“它们在公司项目里到底怎么用、差别在哪里、考试里怎么选”。

因此，需要给第11章增加一份专门的工具与技术说明，用更白话的方式把这些方法讲清楚。

## 要改什么

1. 在第11章新增 `工具与技术/` 目录
2. 新增一份专门讲成本估算方法的白话说明文档
3. 在第11章 `README.md`、`白话拆解.md`、`完整整理.md` 和导航中接入入口
4. 更新当前 OpenSpec 规范，使第11章的工具与技术说明成为正式资产

## 预期效果

- 学习者能直接对比四种成本估算方法的适用场景和区别
- 第11章从“知道术语”提升到“知道该怎么用”
- 上午题和案例题遇到估算方法时，更容易快速判断
