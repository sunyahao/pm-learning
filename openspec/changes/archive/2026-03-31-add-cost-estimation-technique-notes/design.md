# 设计：第11章成本估算方法白话说明

## 文档定位

这份新增文档不替代：

- `完整整理.md` 的系统知识
- `白话拆解.md` 的整章口语化说明

它承担的是“工具与技术单点突破”的角色，专门解决“成本估算方法看定义像懂了，实际不会选”的问题。

## 目录结构

第11章新增：

```text
工具与技术/
  README.md
  成本估算方法白话说明.md
```

## 内容范围

文档至少覆盖：

1. 类比估算
2. 参数估算
3. 自下而上估算
4. 三点估算

每种方法优先采用同一结构：

1. 标准说法
2. 大白话
3. 公司项目里的例子
4. 适合什么时候用
5. 优点
6. 缺点
7. 一句话记忆

## 学习入口

第11章 `README.md` 应新增 `工具与技术` 入口。  
`完整整理.md` 和 `白话拆解.md` 中与估算方法相关的段落，应用链接引导学习者跳转到这份专门说明。
