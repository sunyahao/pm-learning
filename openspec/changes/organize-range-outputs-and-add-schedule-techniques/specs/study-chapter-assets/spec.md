## ADDED Requirements

### 要求：第09章范围管理输出物可以按过程组织样例文件
第09章 `范围管理` 的 `输出物/` 目录可以按过程组织子目录，并通过根目录 `README.md` 作为索引，把样例文件明确挂到 `规划范围管理`、`收集需求`、`定义范围`、`创建WBS`、`确认范围`、`控制范围` 等过程下。

#### 场景：学习者想知道某个范围管理输出物属于哪个过程
- **前提** 学习者进入 `chapters/09-范围管理/输出物/`
- **当** 他打开根目录 `README.md`
- **则** 能按过程看到输出物样例
- **并且** 能进入对应过程子目录查看具体文件

### 要求：第10章进度管理可以提供工具与技术目录
第10章 `进度管理` 可以提供 `工具与技术/` 目录，用于存放高频方法和图示类内容的直观样例。对于 `紧前关系绘图法 PDM` 和 `箭线图法 ADM` 这类仅凭术语难以理解的内容，应优先提供可直接阅读的样例文件。

#### 场景：学习者看到 PDM 和 ADM 但不知道图长什么样
- **前提** 学习者进入 `chapters/10-进度管理/`
- **当** 他打开 `工具与技术/`
- **则** 能看到 `PDM` 和 `ADM` 的具体样例
- **并且** 能通过样例理解它们的表示方式和适用特点
