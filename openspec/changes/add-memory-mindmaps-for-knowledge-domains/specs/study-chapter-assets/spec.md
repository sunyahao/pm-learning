## ADDED Requirements

### 要求：重点项目管理章节如提供重点记忆导航必须服务快速回忆
第08章至第19章如提供 `重点记忆导航.md`，必须围绕本章最值得记忆的主线、过程、关键输出物、高频易混点和答题抓手组织内容，并尽量为关键记忆点提供可直接跳转到对应知识点的链接，用于支持快速回忆，而不是重复长篇讲解。

#### 场景：学习者想在短时间内快速扫过一章主线
- **前提** 学习者已经看过正式讲解材料
- **当** 他在考前或碎片时间复习某章
- **则** 能在该章的 `重点记忆导航.md` 中看到结构化重点列表
- **并且** 该列表能帮助他快速回忆本章重点
- **并且** 关键条目能跳转到对应知识点或样例文件

### 要求：第08章整合管理必须提供重点记忆导航样例
第08章 `整合管理` 必须提供 `重点记忆导航.md`，用于围绕整合目标、七过程、关键输出与易混点形成一份可复习、可跳转的记忆型导航。

#### 场景：学习者想先看整合管理的脑图再复习细节
- **前提** 学习者进入 `chapters/08-整合管理/`
- **当** 他查看本章学习材料
- **则** 能找到 `重点记忆导航.md`
- **并且** 文档能帮助他快速回忆整合管理主线

### 要求：提供重点记忆导航时学习入口应位于核心记忆之后
第08章至第19章如提供 `重点记忆导航.md`，其 `README.md` 学习入口应将该文件放在 `核心记忆.md` 之后，作为章节速记和整体回忆的补充入口。

#### 场景：读者从章节入口寻找速记材料
- **前提** 读者打开任一提供 `重点记忆导航.md` 的章节 `README.md`
- **当** 他查看学习入口
- **则** 能在 `核心记忆.md` 之后看到 `重点记忆导航.md`
