## ADDED Requirements

### 要求：重点项目管理章节必须提供提问文档
第08章至第17章的章节目录下必须提供 `提问文档.md`，用于围绕本章重点概念、过程顺序、关键输出物、高频易混点和案例场景提出结构化问题，服务主动回忆、自测和带学复盘。

#### 场景：学习者在读完重点章节后进行口头自测
- **前提** 学习者已经阅读第08章至第17章中的某一章节学习材料
- **当** 他进入该章节目录，准备做主动回忆或与他人互问
- **则** 能找到 `提问文档.md`
- **并且** 文档中的问题围绕本章重点内容组织
- **并且** 问题既覆盖记忆性提问，也覆盖理解性和案例性追问
