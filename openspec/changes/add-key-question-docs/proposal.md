# 提案：补充十大知识领域重点复盘题单

## 为什么要改

当前仓库已经为第08章至第17章提供了 `精讲.md` 和 `完整整理.md`，可以支持系统阅读与串联理解，但还缺少一类面向主动回忆和口头自测的学习资产。

对于高项备考，尤其是十大知识领域，很多考生不是“看不懂”，而是“看过以后提不出来、问不出来、不会用自己的话回答”。如果仓库只有讲解型文件，学习路径仍然偏输入式，不利于形成可检验的记忆和案例表达能力。

因此，需要为第08章至第17章补充一类新的章节资产，用于围绕重点概念、过程链、易混点和案例场景提出高频问题，帮助学习者：

1. 在阅读后进行主动回忆和自测
2. 按老师带问的节奏复盘重点章节
3. 将零散知识点转化为可口头表达的答案结构

## 本次变更内容

1. 在 `study-chapter-assets` 规范中新增“第08章至第17章必须提供 `复盘题单.md`”的要求
2. 明确 `复盘题单.md` 用于服务重点内容追问、口头自测和案例式复盘
3. 为第08章至第17章补充 `复盘题单.md`
4. 更新对应章节 `README.md`，把 `复盘题单.md` 纳入学习入口
