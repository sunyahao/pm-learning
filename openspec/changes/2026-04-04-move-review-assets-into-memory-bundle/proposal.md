# 提案：将复盘题单与参考作答迁入记忆与复盘目录

## 为什么要改

当前第08章至第19章中的 `复盘题单.md` 和 `参考作答.md` 仍然分散放在章节根目录，而 `记忆与复盘/README.md` 只承担导航作用。这会带来两个问题：

1. “速记、自测、核对”这一组资产没有真正收拢到同一目录，结构表达仍然分裂。
2. 第18章和第19章还没有 `记忆与复盘/` 目录，导致重点章节在复盘资料组织方式上不一致。

用户明确要求：

1. 将所有 `参考作答.md` 和 `复盘题单.md` 移入 `记忆与复盘/`。
2. 通过 OpenSpec 控制这一结构，而不是只做一次性文件移动。

## 本次变更内容

1. 将第08章至第19章的 `复盘题单.md` 和 `参考作答.md` 迁入各章的 `记忆与复盘/` 目录。
2. 为第18章和第19章补齐 `记忆与复盘/README.md`。
3. 更新章节 `README.md`、`核心记忆.md`、`记忆与复盘/README.md` 以及相关文内链接。
4. 更新 OpenSpec 规范，明确复盘题单和参考作答应归入 `记忆与复盘/` 目录，而不再保留在章节根目录。

## 影响范围

- `openspec/specs/study-chapter-assets/spec.md`
- `openspec/changes/2026-04-04-move-review-assets-into-memory-bundle/`
