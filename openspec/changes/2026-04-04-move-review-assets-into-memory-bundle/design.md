# 设计：复盘资料归入记忆与复盘目录

## 设计目标

让重点章节的“速记、自测、核对答案”材料形成真正统一的目录结构，并在章节之间保持一致。

## 关键决策

### `记忆与复盘/` 变为实际承载目录，而不只是导航目录

调整后：

- `核心记忆.md` 继续保留在章节根目录，作为临考速记主入口
- `复盘题单.md` 和 `参考作答.md` 迁入 `记忆与复盘/`
- `记忆与复盘/README.md` 作为本组资产的目录说明与入口

### 第08章至第19章统一使用同一组织方式

当前第08章至第17章已经有 `记忆与复盘/` 目录，第18章和第19章尚未补齐。  
本次统一扩展到第08章至第19章，保证重点章节的复盘资产结构一致。

### 学习入口继续保留，但改为指向新路径

章节 `README.md` 中仍保留“复盘题单”“参考作答”入口，以保持学习顺序清晰；  
只是链接从章节根目录改为 `记忆与复盘/` 下的文件。

## 规范影响

需要修改 `study-chapter-assets` 中关于 `记忆与复盘/`、`复盘题单.md` 和 `参考作答.md` 的现行要求，使其从“根目录文件 + 子目录导航”变为“子目录内实际文件 + 子目录入口”。
