# 设计：增强核心记忆，取消独立重点记忆导航

## 设计目标

保留“临考前快速定位重点”的能力，但避免与 `核心记忆.md` 形成双份速记稿。

## 设计原则

1. 速记入口唯一：章节只保留一份面向临考速记的主入口，默认就是 `核心记忆.md`。
2. 定位优先稳定：优先使用本页跳转和分块结构，不依赖当前环境不稳定的跨文件锚点。
3. 展开阅读外链化：每个重点块下提供对应的展开阅读文件链接，而不是把细节全文搬进速记稿。
4. 内容边界清晰：`核心记忆.md` 负责提炼、定位和分发，不替代 `完整整理.md`、`精讲.md`、`白话拆解.md` 等系统讲解材料。

## 推荐结构

增强型 `核心记忆.md` 可优先包含：

1. 文档用途说明
2. 快速跳转
3. 整章主线
4. 管理基础或最该背的知识块
5. 过程分块速记
6. 高频易混点
7. 案例题抓手
8. 临考前最后看什么

补充要求：

- 重点块应支持跳转到本页具体位置。
- 每个重点块下应有简短记忆点。
- 需要展开时，应链接到 `完整整理.md`、`ITTO输出速看.md`、`输出物/` 或其他对应文件。

## 第08章样例

第08章 `整合管理` 的 `核心记忆.md` 应覆盖：

- 整章主线与七过程口诀
- 整合目标、三个层面、复杂性
- 七过程的最小记忆抓手
- 高频输出物样例入口
- 最易混点
- 案例题答题抓手
