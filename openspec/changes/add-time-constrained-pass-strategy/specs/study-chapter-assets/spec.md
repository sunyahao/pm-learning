## 新增要求

### 要求：核心记忆文件必须服务快速复习
每个章节目录下的 `核心记忆.md` 都必须提炼该章最值得记忆的高频内容，并说明应优先跳转阅读的文件。

#### 场景：上班族利用碎片时间复习某章
- **前提** 学习者只有很短的复习时间
- **当** 他打开某章节目录下的 `核心记忆.md`
- **则** 能快速看到该章最该背的概念、流程或对比点
- **并且** 能看到继续深挖时应跳转的文件路径

### 要求：学习资产应优先服务时间受限考生
当学习资产设计需要在“覆盖面更广”和“复习效率更高”之间权衡时，应优先支持时间受限且以通过考试为目标的学习者。

#### 场景：在有限时间内安排学习内容
- **前提** 学习者每周可投入时间有限
- **当** 仓库提供备考策略、章节导航和记忆材料
- **则** 这些材料优先突出高频考点、核心流程、关键输出物和易错点
