## ADDED Requirements

### 要求：重点项目管理章节必须提供参考作答文档
第08章至第17章的章节目录下必须提供 `参考作答.md`，用于按 `复盘题单.md` 的问题顺序提供参考答案和作答要点，支持自测后核对、结对复习和带学复盘。

#### 场景：学习者在口头回答后核对答案
- **前提** 学习者已经使用 `复盘题单.md` 做过一轮主动回忆
- **当** 他需要核对自己的回答是否完整
- **则** 能在同章节目录下找到 `参考作答.md`
- **并且** 文档中的答案与问题编号保持对应

### 要求：重点项目管理章节学习入口必须先讲解后提问
第08章至第17章的 `README.md` 学习入口中，`复盘题单.md` 必须位于 `精讲.md` 之后，`参考作答.md` 必须位于 `复盘题单.md` 之后，以体现“先系统讲解，再复习思考，再核对答案”的顺序。

#### 场景：读者按章节入口开始学习重点章节
- **前提** 读者进入第08章至第17章任一章节目录
- **当** 他查看 `README.md` 的学习入口
- **则** 能先看到 `完整整理.md` 和 `精讲.md`
- **并且** 之后看到 `复盘题单.md`
- **并且** 在 `复盘题单.md` 之后看到 `参考作答.md`
