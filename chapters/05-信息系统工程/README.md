## 本章定位

- 本章覆盖软件工程、需求工程、架构设计和安全工程，是系统工程基础知识的重点章节。

## 核心考点

- 软件架构风格：数据流、调用/返回、独立构件、虚拟机、仓库风格。
- 需求层次、QFD、需求获取/分析/规格说明/验证与确认。
- 安全工程三维空间、安全服务、ISSE-CMM。

## 高频关键词

- 架构风格、QFD、SRS、数据模型、功能模型、行为模型、安全空间、ISSE-CMM。

## 易错点

- 容易把架构风格的代表模式记混。
- 容易把需求验证与需求确认混为一谈。

## 复盘提示

- 复习时把“软件工程”和“安全工程”分成两块记忆，避免互相干扰。
