## 本章定位

- 本章是组织级项目管理的进阶章节，覆盖项目集管理和项目组合管理，负责把第06章里的层级概念真正拉到战略与治理视角上。

## 核心考点

- 项目集相关角色、项目集绩效域、项目集生命周期管理。
- 项目组合经理职责、项目组合治理机构、项目组合绩效域。
- 高级项目管理更关注战略一致性、效益交付和资源优化。
- 项目、项目集、项目组合在管理对象、价值来源和判断抓手上的差异。

## 高频关键词

- 项目集、项目组合、战略一致性、效益管理、治理、生命周期、优先级排序。

## 易错点

- 容易把项目集和项目组合的目标、对象和价值来源答混。

## 复盘提示

- 做题时先看题干强调“协同收益”还是“战略选择”，再判断是项目集还是项目组合。
- 如果题干出现“多个相关项目需要统一收益实现和依赖协调”，优先往项目集想；如果强调“该投哪些项目、资源怎么分配”，优先往项目组合想。
