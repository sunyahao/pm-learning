## 本章定位

- 本章聚焦信息如何有效传达，是沟通失效、汇报不及时、跨部门协同差等案例问题的标准答案来源。

## 核心考点

- 沟通模型五要素和沟通状态五阶段。
- 书面沟通 5C 原则。
- 规划沟通管理、管理沟通、监督沟通三过程；互动式、推式、拉式沟通。

## 高频关键词

- 编码、解码、噪声、5C、互动式沟通、推式沟通、拉式沟通、沟通管理计划。

## 易错点

- 容易把“信息已发出”误认为“沟通已完成”。
- 容易把沟通方法与干系人管理手段混淆。
