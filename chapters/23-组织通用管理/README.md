## 本章定位

- 本章偏管理实务，覆盖岗位设计、人力资源、招聘、薪酬和流程管理，是组织执行层的基础管理章节。

## 核心考点

- 工作分析方法、岗位设计内容与方法。
- 人力资源战略、供需预测、招聘与录用、薪酬管理。
- 战略执行保障体系、流程管理原则、战略/运行/支持流程分类。
- “组织管理”和“项目资源管理”在对象和层级上的边界。

## 高频关键词

- 岗位设计、人力资源预测、招聘渠道、薪酬调整、流程管理、战略流程、支持流程。

## 易错点

- 容易把第22章的治理层内容和本章的管理执行层内容混淆。

## 复盘提示

- 这章适合按“人-岗-薪-流程”主线复习。
- 做题时如果题干强调组织长期的人才配置和流程体系，不要误答成单个项目团队资源安排。
