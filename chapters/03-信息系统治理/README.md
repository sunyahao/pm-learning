## 本章定位

- 本章聚焦 IT 治理目标、治理层级与治理框架，是组织级治理和标准化题目的基础。

## 核心考点

- IT 治理目标：与业务目标一致、有效利用信息与数据资源、风险管理。
- 三层管理层次：最高管理层、执行管理层、业务与服务执行层。
- IT 治理体系框架、ITSS、COBIT、ISO/IEC 38500。

## 高频关键词

- IT治理、治理闭环、ITSS、COBIT、ISO/IEC 38500、战略一致性、风险管理。

## 易错点

- 容易把“治理”与“管理”混同，忽略治理更偏方向、授权和监督。

## 复盘提示

- 复习时重点盯住“目标、层级、框架、标准”四组关键词。
