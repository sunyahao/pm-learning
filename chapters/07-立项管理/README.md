## 本章定位

- 本章讲项目投资前期工作，重点是项目建议书、可行性研究、项目评估与决策。

## 核心考点

- 立项四阶段：项目建议与立项申请、初步可行性研究、详细可行性研究、评估与决策。
- 项目建议书的定位与核心内容。
- 技术、经济、社会效益、运行环境及法律等可行性分析。

## 高频关键词

- 项目建议书、可行性研究、初步可研、详细可研、项目评估、敏感性分析。

## 易错点

- 容易把项目建议书、可研报告、项目章程放错阶段。
