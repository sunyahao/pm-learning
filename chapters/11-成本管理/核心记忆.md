<a id="core-mainline"></a>
## 一、整章主线

- 成本管理最核心是在花钱之前先估、花钱过程中盯住偏差、花钱之后能预测后续。
- 最核心链条：`成本估算 -> 成本基准 -> 控制成本 -> 成本预测`
- 四过程口诀：`规 -> 估 -> 预 -> 控`
- 这章考试最爱把 `EV / PV / AC` 和 `CPI / SPI / EAC / TCPI` 串着考。

<a id="core-basics"></a>
## 二、管理基础

### 1. 最该背

- `成本估算` 是对单个活动或工作包要花多少钱的估计。
- `成本基准` 是经批准、按时间分配的预算，不含管理储备。
- `项目预算 = 成本基准 + 管理储备`
- `应急储备` 对应已知风险。
- `管理储备` 对应未知风险。

### 2. 挣值最小抓手

- `EV`：挣值
- `PV`：计划价值
- `AC`：实际成本
- `CPI / SPI` 看绩效效率。
- `EAC / ETC / TCPI` 看后续预测和完工要求。

## 三、四过程

### 1. 规划成本管理

- 先把成本怎么估、怎么编预算、怎么监控的规则定下来。
- 高频输出：`成本管理计划`

展开阅读：

- [成本管理计划.md](./输出物/01-规划成本管理/成本管理计划.md)

### 2. 估算成本

- 这一步是在算“每个活动或工作包大概要花多少钱”。
- 高频输出：`成本估算 + 估算依据`

展开阅读：
- [完整整理.md](./完整整理.md)
- [成本估算表.md](./输出物/02-估算成本/成本估算表.md)

### 3. 制定预算

- 这一步把分散估算汇总成正式成本基准。
- 高频输出：`成本基准 + 资金需求`
- 高频提醒：成本基准一旦批准，后续调整要走正式变更控制。

展开阅读：
- [完整整理.md](./完整整理.md)

### 4. 控制成本

- 关键词：看偏差、做预测、控基准。
- 高频输出：`工作绩效信息 + 成本预测 + 变更请求`
- 题干只要出现 EV / PV / AC，优先想到控制成本。

<a id="core-confusions"></a>
## 四、最易混点

- 成本估算 vs 成本基准
  一个是估， 一个是批过的正式预算。
- 应急储备 vs 管理储备
  一个对已知风险，一个对未知风险。
- 成本偏差 vs 进度偏差
  一个看花钱，一个看产出进度。
- CPI vs SPI
  一个看成本效率，一个看进度效率。
- EAC vs TCPI
  一个看预计完工多少钱，一个看后续得达到什么绩效水平。

<a id="core-review"></a>
## 五、案例题抓手

- 先判断题目在问估算、预算，还是已经进入控制成本。
- 若题目给了 EV / PV / AC，就按“偏差 -> 指数 -> 预测”的顺序答。
- 若题目问“为什么超支”，别只写一句成本高，要结合基准、偏差和后续纠偏。
- 若题目问储备动用，先区分是应急储备还是管理储备。
