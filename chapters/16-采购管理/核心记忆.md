<a id="core-mainline"></a>
## 一、整章主线

- 采购管理就是“哪些自己做，哪些找外部做，以及怎么把合同管住”。
- 最核心链条：`采购策略 / SOW / 招标文件 -> 评审谈判 -> 协议 -> 合同控制 / 索赔 / 关闭`
- 三过程口诀：`规 -> 实 -> 控`
- 本章最常和范围、风险、成本、合同争议一起出题。

<a id="core-basics"></a>
## 二、管理基础

### 1. 最该背

- 先会做 `自制或外购分析`。
- `SOW / TOR` 是把采购需求说清楚。
- `RFI`：先了解市场和能力。
- `RFQ`：需求清楚，重点比价格。
- `RFP`：需要卖方提交方案。

### 2. 合同最小抓手

- `FFP`：固定总价，买方风险相对小。
- `CPFF`：成本加固定费用，买方风险相对大。
- 总价合同更适合需求清晰。
- 成本补偿合同更适合范围不够清晰或不确定性高。

## 四、最易混点

- RFI vs RFQ vs RFP
  一个先摸底，一个重报价，一个重方案。
- 总价合同 vs 成本补偿合同
  一个更适合需求清晰，一个更适合不确定性高。
- 检查 vs 审计
  一个偏看交付和履约结果，一个偏看流程和合规性。
- 采购关闭 vs 项目收尾
  一个偏合同闭环，一个偏整个项目闭环。

## 五、案例题抓手

- 如果题目问“采购前缺什么动作”，先补采购管理计划、采购策略、SOW、招标文件。
- 如果题目问“为什么供应商选择失误”，先看供方选择标准和评估过程。
- 如果题目问“合同争议、索赔、交付扯皮”，优先想到控制采购。
- 这章答题很适合按“规划 -> 实施 -> 控制”顺序写。

## 一句话最后记忆

采购管理最该背的不是合同缩写，而是：`先把采购需求和规则定清，再把卖方选对，再把合同履约全过程管住。`
