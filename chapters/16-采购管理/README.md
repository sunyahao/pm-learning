## 本章定位

- 本章讲从外部获取货物或服务的全过程，是合同、招投标和供应商管理的前置章节。

## 核心考点

- 规划采购管理、实施采购、控制采购三过程。
- 一般采购步骤：编 SOW/TOR、估算预算、发布招标、筛选卖方、评标、授予合同。
- 合同管理、索赔、采购审计与履约监督。

## 高频关键词

- SOW、TOR、招标文件、建议书、评标、合同、采购审计、履约。

## 易错点

- 容易把采购过程和合同法律问题混在一起，不分管理过程和法律依据。

## 复盘提示

- 学这章时要和第24章法律法规一起交叉看，但答题时先按采购流程落地。
