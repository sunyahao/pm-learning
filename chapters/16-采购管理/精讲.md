# 精讲：第16章 采购管理

## 来源定位

- 对应 PDF 章节：第16章-采购管理
- 对应页码范围：65-69
- 这份精讲在本章来源基础上补充了教学式串联，重点帮你把“自制或外购、招采、合同履约、监督收尾”讲成一条完整外部采购链。

## 先把这章想明白

采购管理讨论的不是“怎么花钱买东西”这么简单，而是项目为了获得外部货物、服务或成果，要如何决定买不买、怎么买、跟谁买、签什么、怎么监督对方履约。

所以采购管理的本质，是项目和外部供方之间的合作与约束管理。

## 先抓一个总问题：自己做还是外部买

采购管理最前面的核心判断之一，是自制或外购。

- 如果组织自己做，重点在内部资源和能力。
- 如果外部买，重点就转成采购策略、招标、评标、合同和履约控制。

很多项目后面采购出问题，根因其实在前面没有把“为什么要买、买什么、买到什么程度”想清楚。

## 三过程要看成“先规划，再选供方，再控履约”的链

### 1. 规划采购管理

这一过程解决“采购要怎么设计和准备”。

高频输出包括：

- 采购管理计划
- 采购策略
- 招标文件
- 采购工作说明书
- 供方选择标准
- 自制或外购决策

你可以把这一过程理解成采购的方案设计阶段。它要先把下面这些问题讲清：

- 买什么
- 为什么买
- 怎么买
- 怎么选供方
- 后面如何管理供方

这里的采购工作说明书，也就是常说的 SOW 或 TOR，一定要和招标文件、合同区分开。

- SOW/TOR 重点写清采购对象和工作范围。
- 招标文件是面向市场和潜在供方的完整采购文件集合。
- 合同或协议是最终达成的法律性约定。

### 2. 实施采购

这一过程解决“如何从外部市场中选出合适的卖方并正式签约”。

关键输出包括：

- 选定卖方
- 协议
- 变更请求
- 项目文件更新

这一阶段通常会经历：

- 发布采购信息
- 获取卖方响应
- 评审建议书或投标文件
- 按标准比选
- 谈判并授予合同

这里最容易错的是：

- 只写招标，不写评标与选择标准；
- 只写签合同，不写前面的供方筛选；
- 把识别潜在卖方、评审和签约过程顺序答乱。

### 3. 控制采购

这一过程解决“签完合同以后，如何确保供方按约履行”。

关键输出包括：

- 工作绩效信息
- 变更请求
- 项目管理计划更新
- 项目文件更新
- 采购文档更新

控制采购的重点不是“再买一次”，而是看供方是否按合同交付、是否存在偏差、是否需要索赔、变更或纠纷处理。

## 合同类型一定要会分

采购章节常考合同类型。高频分类有：

- 固定总价合同
- 工料合同
- 成本补偿合同

学习合同类型时，关键不是死背名字，而是理解风险主要由谁承担。

- 固定总价合同通常更强调卖方承担较多成本风险。
- 成本补偿合同通常买方承担更多不确定性。
- 工料合同介于两者之间。

## 采购管理和合同法律问题一定要分开

这是本章最容易出错的地方之一。

- 采购管理强调项目管理流程和供方管理动作。
- 合同法律强调法律效力、法条依据和法律责任。

答题时可以交叉引用 [第24章 法律法规与标准规范](../24-法律法规与标准规范/主脉络.md)，但作答主线仍然应该先按采购流程展开。

## 本章最常见的易错点

- 采购步骤前后顺序答乱。
- 把 SOW/TOR、招标文件、协议混成一类。
- 把采购管理和合同法直接合并作答。
- 只写“选供应商”，不写选择标准和履约监督。

## 案例题怎么答

采购类案例题高频问题有：

- 采购策划不完整
- 工作说明书不清
- 供方选择标准不明确
- 合同条款不完善
- 履约监督不严
- 供应商违约后处理失当

答题时最好沿这条线组织：

- 是否先完成采购策划和自制外购分析
- 是否编制采购工作说明书、招标文件和选择标准
- 是否按规范实施采购并确定卖方
- 是否通过协议明确责任、范围和验收要求
- 是否在执行中持续监控供方履约并处理偏差

## 最后的学习提醒

采购管理真正要学会的，不是把“三过程”背出来，而是把这条链走顺：

先决定哪些内容要外部获取 -> 再把采购边界和标准写清 -> 再选择合适供方 -> 再通过合同约束 -> 再持续监督履约

这条链顺了，采购题就不再只是招投标名词堆积。
