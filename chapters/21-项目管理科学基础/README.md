## 本章定位

- 本章主要服务于上午计算题和方法题，是进度、成本等章节的数理基础补充，也是把“公式、结果、管理含义”连起来的总复盘章。

## 核心考点

- 重点是常见计算题型，尤其是进度、成本相关公式与判断。
- 需要形成“公式会用、结果会判、管理含义会说”的完整能力。
- 关键路径、浮动时间、挣值参数、绩效指数和预测公式之间的串联关系。

## 高频关键词

- 计算题、网络图、挣值、预测、量化分析。

## 易错点

- 容易把本章当作独立理论章，实际上它和进度、成本强绑定。

## 复盘提示

- 复习本章时要联动第10章和第11章一起练题。
- 不要把本章当孤立理论章背，必须回到题目里做“算结果 + 说状态 + 提动作”。
