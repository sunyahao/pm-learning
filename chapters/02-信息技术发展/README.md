## 本章定位

- 本章是技术基础章节，主要服务于上午题，覆盖网络、5G、存储、数据库和信息安全基础。

## 核心考点

- 网络分类、OSI 七层、TCP/IP 协议族、802.3 与 802.11。
- SDN 架构、5G 三大场景、DAS/NAS/SAN 区别。
- 数据模型、关系型数据库与 NoSQL、数据仓库、保密性/完整性/可用性。

## 高频关键词

- OSI、TCP/IP、SDN、5G、DAS、NAS、SAN、SQL、NoSQL、数据仓库、CIA。

## 易错点

- 容易把 TCP/IP 分层协议和 OSI 七层硬对应。
- 容易把 DAS、NAS、SAN 的传输对象和适用场景记混。
