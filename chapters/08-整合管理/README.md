## 本章定位

- 本章讲项目经理如何从全局整合各知识域，是案例题和论文题常见统领章节。

## 核心考点

- 整合管理的目标与三个层面：过程、认知、背景。
- 项目管理计划与项目文件的区别。
- 整合管理七过程：制定章程、制定项目管理计划、指导与管理项目工作、管理项目知识、监控项目工作、实施整体变更控制、结束项目或阶段。

## 高频关键词

- 整合管理、项目章程、项目管理计划、项目文件、整体变更控制、项目收尾。

## 易错点

- 容易把配置控制和变更控制混淆。
- 容易把项目管理计划和项目文件的内容范围写反。

## 复盘提示

- 一旦题干涉及跨范围、进度、成本的统筹问题，先想到整合管理。
