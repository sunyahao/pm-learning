# 计算题解题专题

这个目录给“知道第10章、第11章、第21章都很重要，但一到计算题还是发虚”的学习者使用。

它不替代各章节原有的 `完整整理.md`、`精讲.md`、`核心记忆.md`，而是把计算题单独抽出来，按题型连续整理。

## 这个目录解决什么问题

- 关键路径、浮动时间、挣值、预测公式分散在不同章节，不方便连续学。
- 计算题常见的解题顺序、踩坑点和判断语句没有集中在一起。
- 做题时会算，但不会把结果翻译成“项目状态 + 管理动作”。

## 阅读顺序

1. 先看 [关键路径与浮动时间解题套路.md](./关键路径与浮动时间解题套路.md)
2. 再看 [挣值与预测解题套路.md](./挣值与预测解题套路.md)
3. 再看 [综合例题与作答模板.md](./综合例题与作答模板.md)
4. 最后考前用 [临考保底清单.md](./临考保底清单.md) 快速回顾

## 和章节怎么配合

- 进度逻辑和网络图基础，回到 [第10章 进度管理](../../chapters/10-进度管理/README.md)
- 挣值、成本预测和 TCPI，回到 [第11章 成本管理](../../chapters/11-成本管理/README.md)
- 计算总复盘，回到 [第21章 项目管理科学基础](../../chapters/21-项目管理科学基础/README.md)

## 适合什么时候看

- 你已经知道概念，但一到题目就不知道先算什么
- 你会背 `EV / PV / AC`，但不会从结果往下判断
- 你想把上午题里最常见的计算题先集中拿下
