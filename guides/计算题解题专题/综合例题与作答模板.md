# 综合例题与作答模板

## 这一篇怎么用

前两篇解决“会分题型、会算”，这一篇解决“放到综合题和案例题里怎么表达”。

重点不是再多背几个公式，而是练这两个能力：

- 看到题干先知道该走哪条解题线
- 算完以后能把结果翻译成项目判断和管理动作

## 一、先判断题干属于哪一类

### 更像网络图/关键路径题

题干常见信号：

- 活动之间有先后关系
- 问最短工期、关键路径、浮动时间
- 问某活动延误是否影响总工期

### 更像挣值/预测题

题干常见信号：

- 给了 EV、PV、AC 或完成百分比
- 问超支还是节约、超前还是落后
- 问 EAC、ETC、VAC、TCPI

## 二、综合例题 1：延期到底会不会影响总工期

### 题目

某项目当前关键路径为 `B-D-E-G`，活动 `C` 延误 1 天，活动 `D` 延误 2 天。已知：

- 活动 `C` 总浮动时间为 2 天
- 活动 `D` 总浮动时间为 0 天

问：这两项延误分别会不会影响项目总工期？项目经理应优先关注什么？

### 解题思路

1. 先看活动是否在关键路径上
2. 再看延误是否超过浮动时间
3. 最后再说对总工期的影响

### 参考作答

- 活动 `C` 虽然延误 1 天，但其总浮动时间为 2 天，因此当前延误仍在浮动范围内，暂时不一定影响项目总工期。
- 活动 `D` 位于关键路径上，且总浮动时间为 0，延误 2 天将直接增加项目总工期风险。
- 项目经理应优先关注关键路径活动 `D` 的纠偏措施，同时持续观察 `C` 是否继续消耗浮动时间并演变为新的工期风险。

## 三、综合例题 2：算完指数以后怎么说

### 题目

某项目当前：

- `PV = 80`
- `EV = 72`
- `AC = 90`

问：请判断当前项目状态，并说明下一步管理重点。

### 解题思路

1. 先算偏差
2. 再算指数
3. 最后翻译成管理判断

### 计算

- `SV = 72 - 80 = -8`
- `CV = 72 - 90 = -18`
- `SPI = 72 / 80 = 0.90`
- `CPI = 72 / 90 = 0.80`

### 参考作答

- 项目当前进度落后于计划，且成本已出现超支。
- 从绩效指数看，`SPI = 0.90` 表明进度效率偏低，`CPI = 0.80` 表明成本使用效率较差。
- 下一步应重点分析进度落后和成本超支的共同原因，例如关键活动延期、返工、资源利用效率低等，并同步开展预测与纠偏，而不能只盯单一成本数字。

## 四、通用作答模板

### 模板 1：关键路径类

可以直接按这个顺序答：

1. 先指出哪条是关键路径或该活动是否位于关键路径
2. 再说明其浮动时间情况
3. 最后判断是否影响总工期，并补一句管理动作

示例句式：

- 该活动位于关键路径上，总浮动时间为 0，因此其延误将直接影响项目总工期，项目经理应优先采取纠偏措施并更新进度预测。

### 模板 2：挣值类

可以直接按这个顺序答：

1. 先写当前状态
2. 再写依据
3. 最后写趋势和动作

示例句式：

- 当前项目进度落后且成本超支。依据是 `SV < 0`、`CV < 0`，同时 `SPI < 1`、`CPI < 1`。若后续不纠偏，项目将继续面临延期和超预算风险，应及时开展原因分析、预测和调整。

## 五、最常见的失分方式

- 只写数字，不写状态
- 只写状态，不写判断依据
- 只说“加班赶进度”，不先分析关键路径和原因
- 只说“超支”，不说明是否需要预测和纠偏
