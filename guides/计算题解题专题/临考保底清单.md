# 临考保底清单

## 先保这两大类

### 第一类：关键路径与浮动时间

至少记住：

- 关键路径 = 从起点到终点总持续时间最长的路径
- 关键路径活动最不能拖
- 总浮动时间看是否影响总工期
- 自由浮动时间看是否影响紧后活动

### 第二类：挣值与预测

至少记住：

- `PV` 看计划
- `EV` 看成果
- `AC` 看花费
- `SPI < 1`：进度落后
- `CPI < 1`：成本超支
- `EAC > BAC`：照当前趋势会超预算

## 考场上的最短解题顺序

### 碰到网络图题

1. 先排逻辑
2. 再正推
3. 再逆推
4. 最后找关键路径和浮动时间

### 碰到挣值题

1. 先认 `PV / EV / AC`
2. 再算 `SV / CV`
3. 再看 `SPI / CPI`
4. 最后判断 `EAC / ETC / VAC`

## 最短作答句式

### 关键路径题

- 该活动位于关键路径上，且总浮动时间为 0，因此其延误将直接影响项目总工期。

### 挣值题

- 当前项目进度落后且成本超支，依据是 `SV < 0`、`CV < 0`，同时 `SPI < 1`、`CPI < 1`。

## 最容易混的点

- `EV` 和 `AC`
- `总浮动时间` 和 `自由浮动时间`
- `CPI` 和 `SPI`
- `EAC` 和 `ETC`
